# 🚀 Full Stack Project – BFHL API

This project is a Full Stack Application built as part of a coding challenge.  
It processes hierarchical relationships from input data and returns structured insights such as trees, cycles, and summaries.

---

## 📌 Features

- Accepts node relationships like `A->B`
- Validates input format
- Removes duplicate edges
- Detects cycles in graph
- Builds hierarchical tree structure
- Calculates depth of each tree
- Provides summary of trees and cycles
- Interactive frontend UI to test API
